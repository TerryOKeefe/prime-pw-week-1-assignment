# 8 - Why would you run the command 'git status'?
The command displays the state of the working directory. It lets you see what has been staged, which have not. Status can be seen with red or green colored text.
# 9 - What does the command 'git add .' do?
The command adds a change in the directory to the staging area. It tells Git to include the updates to that particular file when the commit command is used.
